## What Is This?

**human-ai-playbook** is a continuously evolving human-AI collaboration methodology with three deliverables:

| # | Deliverable | Audience | Core Value |
|---|-------------|----------|------------|
| 1 | **Methodology Docs** | Humans | Build systematic understanding: why, what, and how |
| 2 | **Agent System** | AI (and humans through AI) | Executable collaboration protocols that guide each phase |
| 3 | **Self-Evolution Engine** | The project itself | Keep methodology and Agents current with AI advancements |

## Core Principles (Preview)

1. **Requirements First** — When requirements are unclear, AI's execution power becomes a disaster
2. **Context Engineering > Prompt Engineering** — Managing what AI sees matters more than how you ask
3. **Think, Then Plan, Then Execute** — Enforce phases; human confirms before advancing
4. **Divide and Conquer** — Break large tasks into small, verifiable, rollback-able steps
5. **Verification Loop Is Non-Negotiable** — All AI output must have a verification mechanism
