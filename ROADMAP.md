# Roadmap

## Vision

Cover the full project lifecycle with human-AI collaboration methodology and executable agents.

```
Project Lifecycle:  Kickoff -> Daily Collab -> Retrospective -> Transfer/End
                      |            |              |                |
Agents:            Kickoff     Async-Comm    Retrospective    Persona-Transfer
                   Agent       Agent         Agent            Agent
                   * MVP       o v2          o v2             o v3

Always-on:         Meta Agent (constitution layer) * MVP
                   Collaboration Quality Diagnosis Agent o v3
                   Requirements Clarification Agent o v2
```

`*` = MVP scope, `o` = future version

---

## Development Phases

### Phase 1: Foundation
- README + ROADMAP
- Core principles documentation
- Meta Agent constitution
- Research materials

**Done when**: A reader can understand the project's purpose and methodology.

### Phase 2: Core Capability
- Kickoff Agent (full implementation)
- Template fragment library
- Self-test 3+ times

**Done when**: Generate a usable CLAUDE.md in 30 minutes.

### Phase 3: Evolution Engine
- Full self-evolution pipeline
- GitHub Actions automation
- First 2 weekly digests

**Done when**: Weekly automated digest + update suggestions.

### Phase 4: Content and Polish
- All 6 methodology documents
- English translations
- 3-5 user tests

**Done when**: 3+ positive external user feedback.

### Phase 5: Launch
- Launch article
- Multi-platform promotion
- Contributing guide
